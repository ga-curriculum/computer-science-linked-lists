# ![Computer Science - Linked Lists](./assets/hero.png)

## Overview

Linked lists are a foundational data structure that many other, more complex structures use as their basis.

### Prerequisites

- Big O Notation

### Learning Objectives

By the end of this module, learners will be able to:

- Define a linked list.
- Explain how a node stores data.
- Identify situations where you use a linked list instead of an array.
- Build a singly linked list.

## Content

There are two components to this module:

1. Lesson: [Linked Lists](./lesson/README.md)
2. Exercise: [Implementing a Linked List](https://git.generalassemb.ly/modular-curriculum-all-courses/linked-lists-starter-code)

## Solution Code 

- Exercise Solution: [Linked List](./solution/README.md)

### Time to Complete

2.5 hours total:

- 0.5 hour - Linked Lists Lesson
- 2 hour - Implementing a Linked List Exercise
